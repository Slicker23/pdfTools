Thank you for your interest in contributing.

pull request, issue comment, or patch, you agree that:

1. Your contribution is your original work (or you have the right to submit it).
2. You grant Alexandru Bucur a perpetual, worldwide license to use, modify, and

3. Your contribution will be part of this open-source project; you retain

   AGPL-3.0 when merged.

If you need a different arrangement (e.g. corporate CLA), contact
**bucur.alexandru0106@gmail.com** before contributing large changes.

## How to contribute

1. Open an issue to discuss significant changes before large PRs.
2. Fork the repository and create a feature branch.
3. Run tests: `npm test` and `npm run build:engine`.
4. Open a pull request with a clear description of the change.

## Code of conduct

Be respectful and constructive. Maintainers may reject contributions that do not
fit the project direction.
