# PDF engine fonts

DejaVu Sans TTF files used for Unicode text in `apply.py`.

Run `bash scripts/setup-pdf-engine.sh` to copy fonts from the system
(`dejavu-sans-fonts` package) or fall back to bundled Noto Sans.

Expected files:

- `DejaVuSans.ttf`
- `DejaVuSans-Bold.ttf`
- `DejaVuSans-Oblique.ttf`
- `DejaVuSans-BoldOblique.ttf`
